# 🚚 Logistics MIS Dashboard | Excel • SQL • KNIME • Reporting Automation

## 📌 Project Overview

Designed and developed an end-to-end Logistics Management Information System (MIS) Dashboard to monitor shipment operations, revenue performance, carrier efficiency, warehouse utilization, and delivery KPIs across pan-India logistics operations.

The solution automated reporting workflows and provided management with actionable insights through interactive dashboards and automated performance reports.

---

## 🎯 Business Objectives

* Automate logistics reporting processes
* Monitor operational and financial KPIs
* Track shipment delivery performance
* Analyze carrier and warehouse efficiency
* Improve business decision-making through data-driven insights
